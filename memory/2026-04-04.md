## 2026-04-04 (토) - 스마트 세탁공장 구독 관리 및 결제 시스템 고도화

### 주요 업데이트
- **결제 승인 프로세스 도입:**
  - 공장 대표가 결제 신청 시 즉시 적용되지 않고 `pendingPayments` 리스트에 저장됨.
  - 총괄 관리자(플랫폼 마스터)가 결제 신청 내역을 [결제 승인 대기 목록]에서 확인 후 승인해야 최종적으로 구독 상태 및 만료일이 업데이트되도록 변경.
  - 관리자 화면에서 '세금계산서 발행 요청' 여부 확인 가능.
- **결제 UI/UX 고도화:**
  - 결제 기간 선택 기능 추가: 1개월, 3개월(5%), 6개월(10%), 12개월(20%) 할인 정책 적용.
  - 부가세(VAT) 자동 계산 기능: '세금계산서 발행 요청' 체크 시 10% 부가세가 자동으로 총금액에 합산되며, 화면에 명시적으로 표시.
  - 결제 진행 중 상태 안내: 결제 신청 후 승인 대기 중인 공장이 재로그인 시, '결제 승인 대기 중' 전용 팝업이 뜨도록 로직 추가.
  - 공장별 등록된 은행 계좌 정보(`bankInfo`)를 결제 화면에 노출하여 수동 계좌이체 유도.

### 향후 고려 사항
- 관리자가 승인한 결제 내역을 승인 완료/발행 완료 등의 상태로 기록 관리하는 기능 고려 가능.
- 현재는 `localStorage` 기반 상태 관리 중으로, 서버(Supabase) 동기화 시 데이터 정합성 지속 확인 필요.
