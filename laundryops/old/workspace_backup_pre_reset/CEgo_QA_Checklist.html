CEgo Platform QA 테스트 체크리스트

프로젝트명: CEgo 세탁공장 SaaS 플랫폼
문서버전: v04_Final_QA
출력일자: 2026-03-23

※ 본 문서는 정식 서비스 런칭 전, 시스템의 각 권한별 기능 작동 여부를 최종 점검하기 위한 테스트 가이드입니다.

1. 👑 플랫폼 총괄 관리자 (SuperAdmin)

자동 로그인: 드롭다운 '플랫폼 총괄 관리자' 선택 시 ID/PW(admin) 자동 입력 확인
GMV 집계: '운영중' 상태인 모든 공장의 합계 매출이 메인 대시보드에 정확히 반영되는가?
월별 조회: '조회 월' 필터 변경 시 해당 월의 매출 순위(TOP 10)가 즉시 갱신되는가?
상태 관리: 공장을 '미운영'으로 변경 시 목록에서 회색 처리 및 통계 제외가 정상 작동하는가?
데이터 이관: [테스트 데이터 내보내기] 시 JSON 파일 다운로드 및 [가져오기] 시 데이터 복원 확인

2. 🏢 세탁공장 대표 (Admin)

경영 분석: 최근 6개월 매출 추이 그래프 노출 및 호텔별 개별 추이 필터링 작동 확인
성장률: 전월 동기 대비 매출 증감률(%)이 정확한 로직으로 계산되는가?
정보 수정: 호텔 파트너의 아이디/비밀번호 변경 및 저장 기능 확인
단가 관리: 개별 호텔의 품목 추가 및 맞춤 단가 설정 시 즉각적인 저장 여부
리포트 출력: 기간 검색 후 [PDF 리포트 저장] 시 공급가액, 부가세, 합계 금액 노출 확인

3. 👕 세탁공장 현장직원 (Staff)

입력 효율: 수량 입력창에서 Enter 키 입력 시 아래 품목으로 자동 포커스 이동 확인
중복 처리: 이미 발행된 날짜/호텔 선택 시 기존 데이터 로드 및 수정 모드 전환 확인
발행 기록: 명세서 저장 즉시 하단 '최근 발행 목록'에 리스트업 및 상세보기 작동 확인

4. 🏨 호텔 파트너 (Hotel)

비용 모니터링: 전용 대시보드 상단에 이번 달 누적 이용 금액이 정확히 노출되는가?
통계 분석: 우리 호텔이 가장 많이 맡긴 세탁물(도넛 차트) 비중이 정상 시각화되는가?
명세서 수신: 공장에서 발행한 명세서를 실시간으로 확인하고 부가세 포함 상세 내역 보기 가능 여부

5. ⚙️ 시스템 공통 (Common)

키보드 지원: 패스워드 입력 후 Enter 키로 로그인 시도 시 정상 접속 확인
세션 종료: 로그아웃 버튼 클릭 시 초기 로그인 화면으로 안전하게 리다이렉트 확인
데이터 보존: 브라우저 새로고침 및 재시작 시에도 LocalStorage 데이터 유지 확인

* 테스트 중 발견된 오류나 개선사항은 비서 '마루'에게 즉시 전달하여 수정 조치하시기 바랍니다.
